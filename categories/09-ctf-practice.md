## 🎮 CTF Platforms

### **TryHackMe**
- **URL**: https://www.tryhackme.com/
- **Description**: Interactive cybersecurity training platform with structured learning paths, rooms (labs), and CTF challenges for all levels
- **Type**: Training Platform / CTF
- **Level**: 🟢 Beginner - 🔴 Advanced
- **Cost**: Free (Premium available)
- **Features**:
  - 1000+ rooms (labs) covering various topics
  - Structured learning paths:
    - Security Fundamentals
    - Red Team Operations
    - Blue Team Defense
    - Cloud Security
    - Web Security
  - Integrated attack boxes
  - Progress tracking
  - Achievements and certificates
  - Community events and competitions
- **Time Commitment**: 30 minutes - 4 hours per room
- **Best For**: Beginners to intermediate learners with structured learning

### **HackTheBox**
- **URL**: https://www.hackthebox.com/
- **Description**: Platform providing vulnerable machines and applications for penetration testing practice and skill development
- **Type**: CTF / Practice Labs
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Cost**: Freemium (Free accounts have limited access)
- **Features**:
  - 200+ vulnerable machines (Linux, Windows, misc)
  - Web applications for testing
  - API pentest labs
  - Forensics challenges
  - Mobile security labs
  - Real-world scenarios
  - Monthly retired machines
- **Machine Types**:
  - Easy (entry-level)
  - Medium (intermediate)
  - Hard (advanced)
  - Insane (extreme)
- **Best For**: Intermediate to advanced hands-on practice

### **HackTheBox Academy**
- **URL**: https://academy.hackthebox.com/
- **Description**: Structured cybersecurity academy with interactive modules, labs, and comprehensive learning paths
- **Type**: Academy / Training
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Cost**: Free (Premium available)
- **Modules Include**:
  - Web Security (SQLi, XSS, CSRF, etc.)
  - Penetration Testing
  - Privilege Escalation (Windows & Linux)
  - Active Directory & Enumeration
  - Reverse Engineering
  - Cryptography
  - Forensics
  - Network Security
- **Features**:
  - Interactive labs for each module
  - Hands-on exercises
  - Video explanations
  - Module certificates
  - Difficulty scaling
- **Best For**: Structured learning with deep-dive modules

### **PicoCTF**
- **URL**: https://www.picoctf.com/
- **Description**: Educational capture-the-flag competition designed for beginners with progressive difficulty levels
- **Type**: CTF / Competition
- **Level**: 🟢 Beginner - 🟡 Intermediate
- **Cost**: Free
- **Features**:
  - 100+ challenges
  - Beginner-friendly difficulty progression
  - Categories:
    - Cryptography
    - Forensics
    - General Skills
    - Reverse Engineering
    - Web Exploitation
  - Hints available
  - Community support
  - Annual competition
- **Time Commitment**: 30 minutes - 2 hours per challenge
- **Best For**: First-time CTF participants and beginners

### **CtfTime**
- **URL**: https://ctftime.org/
- **Description**: CTF aggregator and archive providing information about ongoing and past CTF competitions worldwide
- **Type**: Aggregator / Calendar
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Cost**: Free
- **Features**:
  - CTF calendar and schedule
  - Team rankings
  - Event archives
  - Writeups from past CTFs
  - Competition information
- **Best For**: Finding and tracking CTF competitions

### **OWASP WebGoat**
- **URL**: https://owasp.org/www-project-webgoat/
- **Description**: Deliberately insecure web application designed for teaching web security vulnerabilities and exploitation
- **Type**: Vulnerable Application / Training
- **Level**: 🟢 Beginner - 🟡 Intermediate
- **Cost**: Free
- **Lessons Cover**:
  - Injection
  - Authentication
  - Authorization
  - Sensitive Data Exposure
  - XXE (XML External Entity)
  - Broken Access Control
  - Security Misconfiguration
  - XSS (Cross-Site Scripting)
  - Insecure Deserialization
- **Best For**: Learning OWASP Top 10 vulnerabilities

## 🔓 Vulnerable Applications

### **DVWA (Damn Vulnerable Web Application)**
- **URL**: https://github.com/digininja/DVWA
- **Description**: Intentionally vulnerable PHP web application for practicing web security testing
- **Type**: Vulnerable Application
- **Level**: 🟢 Beginner - 🟡 Intermediate
- **Cost**: Free / Open Source
- **Vulnerabilities**:
  - SQL Injection
  - Authentication bypass
  - File inclusion
  - File upload
  - Insecure CAPTCHA
  - Cross-Site Scripting (XSS)
  - Cross-Site Request Forgery (CSRF)
  - Using known vulnerable components
- **Setup**: Docker, VirtualBox, or direct installation
- **Best For**: Learning OWASP vulnerabilities

### **OWASP Juice Shop**
- **URL**: https://owasp.org/www-project-juice-shop/
- **Description**: Modern vulnerable web application containing OWASP Top 10 vulnerabilities and other security flaws

- **Cost**: Free / Open Source
- **Features**:
  - 100+ challenges
  - Realistic e-commerce application
  - Modern JavaScript framework
  - Docker support
  - Built-in scoring
- **Vulnerabilities Cover**:
  - Injection
  - Broken Authentication
  - Sensitive Data Exposure
  - XML External Entities
  - Broken Access Control
  - Security Misconfiguration
  - XXE
  - XSS
  - Insecure Deserialization

## 🚀 Getting Started

### **For Complete Beginners**
1. Start with PicoCTF or TryHackMe Pre Security path
2. Learn Linux basics (OverTheWire Bandit)
3. Learn web security (PortSwigger Web Security Academy)
4. Move to DVWA or OWASP WebGoat
5. Progress to more complex CTF platforms

### **For Intermediate Learners**
1. Start with HackTheBox - Starting Point
2. Practice with OWASP Juice Shop
3. Take TryHackMe learning paths
4. Join CTF competitions on CtfTime
5. Work through HackTheBox Academy modules

### **For Advanced Learners**
1. Target HackTheBox hard/insane machines
2. Participate in competitive CTF events
3. Create custom labs and challenges
4. Contribute to the community with writeups
5. Specialize in specific areas (forensics, crypto, etc.)

### **Recommended Practice Schedule**

**Weekly Practice**
- 3-4 hours of structured learning
- 2-3 hours of hands-on lab practice
- 1 hour of CTF challenges
- 1 hour of reading/research
