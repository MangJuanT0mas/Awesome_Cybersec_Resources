## 🟢 Free Training Platforms

These platforms offer free access to quality cybersecurity training without requiring payment.

### **Cisco Skills for All**
- **URL**: https://skillsforall.com/course
- **Description**: Free networking and cybersecurity courses from industry leader Cisco, including networking fundamentals, cybersecurity basics, and IT essentials
- **Type**: Learning Platform
- **Level**: 🟢 Beginner - 🟡 Intermediate
- **Cost**: Free
- **Certifications**: Yes (Skills for All badges)
- **Duration**: Self-paced
- **Topics**: Networking, Cybersecurity Fundamentals, IT Essentials

### **PortSwigger Web Security Academy**
- **URL**: https://portswigger.net/web-security
- **Description**: Free comprehensive web application security course with interactive labs covering OWASP Top 10, advanced web vulnerabilities, and real-world attack scenarios
- **Type**: Course with Labs
- **Level**: 🟢 Beginner - 🔴 Advanced
- **Cost**: Free
- **Certifications**: Free certificates available
- **Duration**: 50+ hours of content
- **Topics**: Web App Security, API Security, Client-Side Attacks, Authentication Bypasses, SQL Injection, XSS, CSRF

### **Google SecOps (Chronicle Security)**

- **Type**: Training & Platform
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Cost**: Free (Platform varies)

- **Duration**: Self-paced
- **Topics**: Security Operations, Threat Hunting, Incident Response, Log Analysis

### **Qualys Training**

## 🟡 Free Certifications

Certifications that can be obtained at little to no cost.

### **Cisco Certified CyberOps Associate**
- **URL**: https://www.cisco.com/c/en/us/training-events/training-certifications/certifications/associate/ccna-cyberops.html
- **Exam Cost**: ~$330 USD
- **Prerequisites**: Networking knowledge recommended
- **Level**: 🟡 Intermediate
- **Focus**: Cybersecurity operations, threat analysis, security monitoring
- **Study Resources**: Free and paid materials available

### **CompTIA Security+**
- **URL**: https://www.comptia.org/certifications/security
- **Exam Cost**: ~$380 USD (vouchers often available at discount)
- **Prerequisites**: Network+ recommended (or equivalent experience)
- **Level**: 🟡 Intermediate
- **Focus**: General cybersecurity, threat analysis, compliance
- **Study Resources**: Many free resources available online
- **Industry Recognition**: Highly recognized globally

### **EC-Council Certified Ethical Hacker (CEH)**
- **URL**: https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/
- **Exam Cost**: ~$200-400 USD
- **Prerequisites**: 2+ years cybersecurity experience or CEH training
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Focus**: Penetration testing, ethical hacking, attack methodologies
- **Study Resources**: Official and community resources available
- **Lab Access**: Some bundled lab access

### **Google Cloud Security Engineer Associate**
- **URL**: https://cloud.google.com/certification/cloud-security-engineer
- **Exam Cost**: ~$200 USD
- **Prerequisites**: GCP and security knowledge
- **Level**: 🟡 Intermediate
- **Focus**: Google Cloud security, identity, access management
- **Study Resources**: Free Google training available

### **Microsoft Security Engineer Associate (AZ-500)**
- **URL**: https://learn.microsoft.com/en-us/certifications/security-engineer/
- **Exam Cost**: ~$165 USD
- **Prerequisites**: Azure fundamentals recommended
- **Level**: 🟡 Intermediate
- **Focus**: Azure security, identity, compliance
- **Study Resources**: Free Microsoft Learn modules

### **AWS Certified Security - Specialty**
- **URL**: https://aws.amazon.com/certification/certified-security-specialty/
- **Exam Cost**: ~$300 USD
- **Prerequisites**: AWS fundamentals recommended
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Focus**: AWS security, identity, encryption, compliance
- **Study Resources**: Free and paid materials available

## 🔴 Industry-Recognized Certifications

Premium certifications that are highly regarded in the industry.

### **OSCP (Offensive Security Certified Professional)**
- **URL**: https://www.offsec.com/courses/pen-200/
- **Course Cost**: ~$999 USD (includes 30 days lab access)
- **Exam Cost**: ~$165 USD
- **Prerequisites**: Penetration testing fundamentals
- **Level**: 🔴 Advanced
- **Focus**: Hands-on penetration testing, exploitation, privilege escalation
- **Lab Experience**: 30 days of lab access included
- **Industry Recognition**: ⭐⭐⭐⭐⭐ Highly respected

### **GIAC Security Essentials (GSEC)**
- **URL**: https://www.giac.org/certification/security-essentials-gsec
- **Course Cost**: ~$8,000+ (SANS course)
- **Exam Cost**: Included in course
- **Prerequisites**: Security fundamentals
- **Level**: 🟡 Intermediate
- **Focus**: General security operations, threat analysis
- **Industry Recognition**: ⭐⭐⭐⭐ Very respected

### **GIAC Certified Incident Handler (GCIH)**
- **URL**: https://www.giac.org/certification/certified-incident-handler-gcih
- **Course Cost**: ~$8,000+ (SANS course)
- **Exam Cost**: Included in course
- **Prerequisites**: Security operations experience
- **Level**: 🟡 Intermediate - 🔴 Advanced
- **Focus**: Incident response, handling, recovery
- **Industry Recognition**: ⭐⭐⭐⭐ Highly respected

### **CEH (Certified Ethical Hacker) - Advanced**
- **URL**: https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/
- **Course Cost**: ~$1,000-3,000
- **Exam Cost**: ~$400
- **Prerequisites**: CEH or equivalent
- **Level**: 🔴 Advanced
- **Focus**: Advanced penetration testing, social engineering
- **Industry Recognition**: ⭐⭐⭐⭐ Well recognized

### **CISSP (Certified Information Systems Security Professional)**
- **URL**: https://www.isc2.org/certifications/cissp
- **Exam Cost**: ~$749
- **Prerequisites**: 5+ years security experience or equivalent education
- **Level**: 🔴 Advanced
- **Focus**: Security governance, risk management, comprehensive security topics
- **Industry Recognition**: ⭐⭐⭐⭐⭐ Gold standard in security management
- **Note**: Requires sponsorship and experience verification

## 🎯 Learning Paths by Role

### **Path 1: Red Team Operator**
**Timeline**: 6-12 months

1. **Start** (Month 1-2)
   - Cisco Skills for All - Networking Fundamentals
   - PortSwigger Web Security Academy

2. **Intermediate** (Month 3-6)
   - HackTheBox Academy - Penetration Testing Path
   - CompTIA Security+ (optional but recommended)

3. **Advanced** (Month 7-12)
   - OSCP Preparation
   - RedTeam-focused CTFs
   - Real-world lab practice

4. **Certification Target**: OSCP

### **Path 2: Blue Team Defender**
**Timeline**: 6-12 months

1. **Start** (Month 1-2)
   - Cisco Skills for All
   - Splunk Free Courses

2. **Intermediate** (Month 3-6)
   - TryHackMe - Blue Team Path
   - Blue Cape Security DFIR Course

3. **Advanced** (Month 7-12)
   - Google SecOps (Chronicle)
   - Incident Response Advanced Training
   - Threat Hunting Specialization

4. **Certification Target**: GCIH or Splunk Certified Administrator

### **Path 3: Cloud Security Engineer**
**Timeline**: 6-9 months

1. **Start** (Month 1-2)
   - Microsoft Learn - Azure Fundamentals
   - Cisco Skills for All

2. **Intermediate** (Month 3-6)
   - Microsoft Security Engineer Associate (AZ-500)
   - Google Cloud Security Engineer

3. **Advanced** (Month 7-9)
   - AWS Certified Security - Specialty
   - Advanced cloud security specialization

### **Path 4: SOC Analyst**
**Timeline**: 3-6 months

1. **Start** (Month 1)
   - Cisco Skills for All
   - Splunk Free Courses (Fundamentals)

2. **Intermediate** (Month 2-4)
   - TryHackMe - SOC Analyst Path
   - Qualys Training

3. **Advanced** (Month 5-6)
   - Google SecOps
   - Threat Intelligence Analysis

4. **Certification Target**: Cisco CyberOps Associate or Splunk Certified User

## 💡 Tips for Choosing Training

1. **Assess Your Current Level**: Start with beginner courses even if you have experience
2. **Choose Your Specialization**: Decide between offensive, defensive, or cloud security
3. **Consider Time Investment**: Mix free and paid options based on your schedule
4. **Hands-on Practice**: Always choose platforms with labs over lectures only
5. **Community Support**: Pick programs with active communities for help
6. **Industry Recognition**: Consider certifications valued in your target job market
7. **Budget Wisely**: Start with free resources, then invest in quality certifications
